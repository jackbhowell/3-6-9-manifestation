# 📱 App Name:

3-6-9 Manifestation Companion

---

# 🎯 Core Goal:

Create a calm, minimal mobile app that helps users complete daily affirmations using the 3-6-9 manifestation method over a 33 or 45 day cycle.

---

# ⚙️ Technical Requirements:

* Use **Expo (React Native)**
* Use **functional components + hooks**
* Use **AsyncStorage** for local persistence
* Use **Expo Notifications API** for reminders
* Use **Expo AV** for sound playback
* Keep dependencies minimal
* Clean, modular file structure

---

# 📁 Required Screens:

## 1. OnboardingScreen

* Let user:

  * Select cycle length (33 or 45 days)
  * Enter main intention (optional)
  * Set notification times:

    * Morning
    * Afternoon
    * Evening
* Save settings locally
* Navigate to HomeScreen

---

## 2. HomeScreen

* Display:

  * Current day (e.g., Day 5 of 33)
  * Progress for today:

    * Morning (0/3)
    * Afternoon (0/6)
    * Evening (0/9)
* Buttons for each session:

  * “Start Morning”
  * “Start Afternoon”
  * “Start Evening”

---

## 3. AffirmationScreen

* Dynamic based on session type:

  * Morning → 3 inputs
  * Afternoon → 6 inputs
  * Evening → 9 inputs

### Features:

* Input fields with counter
* Prevent submission until all fields are filled
* Save affirmations to storage (by day + session)
* On completion → navigate to ReflectionScreen

---

## 4. ReflectionScreen

* 60-second countdown timer

### UI:

* Large animated circle (breathing/pulsing effect)
* Minimal design (dark calming background)

### Features:

* Countdown from 60 → 0
* Play soft chime sound at end
* Button: “Complete Session” → return to HomeScreen

---

## 5. ProgressScreen

* Calendar or list view of days
* Show:

  * Completed sessions per day
* Simple and clean UI

---

# 🔔 Notification System:
